## RubyVE site

Repo for the RubyVe community website. Powered by Jekyll and Github-Pages.

### Installation

- Use Ruby > 1.9.2
- Install Bundler `gem install bundler`
- Use bundler to install dependencies `bundle install`
- Run the server `bundle exec jekyll serve -w`
- Happy contributing!
